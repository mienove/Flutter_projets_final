# Projet To-Do List Flutter

Application mobile pour gérer des tâches et notes, avec gestion des utilisateurs.

## Fonctionnalités
- Connexion et inscription
- Gestion des tâches (ajouter, modifier, supprimer)
- Déconnexion et navigation
- Stockage des données avec SQLite

## Structure du projet
- lib/
  - taches/
    - connexion.dart
    - inscription.dart
    - accueil.dart
    - traitement.dart
    - modeltac.dart
    - modelutilisateur.dart
  - Data/
    - DatabaseManager.dart
- assets/images/ → logos et icônes

## Instructions
1. Cloner le projet
2. Exécuter `flutter pub get`
3. Lancer sur un émulateur ou appareil
